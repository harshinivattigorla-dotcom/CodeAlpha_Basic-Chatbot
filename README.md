# Basic Chatbot 🤖

## Overview
This project is a simple rule-based chatbot developed using Python. It responds to predefined user inputs and demonstrates the use of conditional statements, loops, functions, and input/output operations.

## Features
- Responds to greetings like "hello"
- Answers "how are you"
- Ends the conversation when the user types "bye"
- Handles unknown inputs gracefully

## Technologies Used
- Python

## Concepts Used
- Functions
- if-elif-else statements
- while loops
- User Input/Output
- String handling

## Sample Inputs
- hello
- how are you
- bye

## Sample Output
You: hello
Bot: Hi!

You: how are you
Bot: I'm fine, thanks!

You: bye
Bot: Goodbye!
